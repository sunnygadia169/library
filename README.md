# Book Library
 
Library is a solution for online book selling system where customers can be onboarded, books can be added, orders can be placed.

## Tech Stack ##

* Java 11
* Maven 3.6.1
* MongoDB 4.2.15
* Spring Boot 2.5.4
* Springfox for Swagger 2.6.1
* Docker

## Swagger UI ##
* http://localhost:8080/swagger-ui.html
